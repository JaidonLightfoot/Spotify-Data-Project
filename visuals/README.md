## Tableau Dashboard

This interactive dashboard explores Spotify track popularity, artist reach, genre trends, and release strategies across decades.

### Key Insights
- Popularity trends over time and by decade
- Correlation between artist followers and track popularity
- Genre performance using heatmaps
- Top artists and tracks by reach and popularity
